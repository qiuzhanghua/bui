# BUI(Bulma UI)

### for svelte

## Initialize

```bash
yarn create svelte bui

cd bui
```

## Developing

```bash
yarn dev

yarn dev --open

yarn add -D svelte2tsx

```

## Building

To create a production version of your app:

```bash
yarn build
```

## Package

```bash
yarn package
```

## Publish

```bash
# npm login

npm publish --access public
```

---

## Story Book

```bash
yarn storybook
```

or

```bash
# run this for new version
yarn build-storybook
# and then
http-server storybook-static
```
